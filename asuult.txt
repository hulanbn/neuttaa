Version Control System гэж юу вэ?

Version Control System (VCS) нь програмын эсвэл бичиг баримтын өөрчлөлтийг хянаж,
хөгжүүлэлтийн түүхийг хадгалж байдаг хэрэгсэл юм. Энэ нь олон хөгжүүлэгч нэг зэрэг нэг төсөл дээр ажиллах 
боломжийг олгодог. Мөн өмнөх хувилбаруудыг буцаах, өөрчлөлтийн түүхийг харах, өөрчлөлтийг хялбархан хянах
 боломжийг олгодог.


Git гэж юу вэ?

Git бол түгээмэл хэрэглэгддэг Version Control System бөгөөд тархмал хэлбэрийн систем гэж үздэг. 
Энэ нь төсөл дээрх өөрчлөлтийг хянах, кодыг хуваалцах, буцаах, салбарлах зэрэг олон функцтэй. 
Git нь олон хөгжүүлэгч зэрэг ажиллахад хамгийн тохиромжтой бөгөөд офлайн горимд ч ашиглах боломжтой.


Github болон gitlab гэж юу вэ? Ямар ялгаатай вэ?

GitHub болон GitLab нь хоёулаа Git-ийн хадгалсан төслүүдийг онлайнаар удирдах, 
бусадтай хуваалцдаг платформууд юм.

Ялгаа:

GitHub-г Microsoft компани эзэмшдэг бөгөөд илүү олон нийтэд нээлттэй байдаг ба, 
хөгжүүлэгчдийн дунд түгээмэл ашиглагддаг.
GitLab нь илүү хувийн болон дотоод төслүүдийг дэмждэг.
Түүнчлэн, GitLab-г өөрийн сервер дээр суулгаж, 
ашиглах боломжтой тул байгууллагуудын дотоод сүлжээнд тохиромжтой байдаг.
